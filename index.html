Key Features

BloodLine offers a comprehensive set of features to make blood donation easier

Secure Authentication

Create accounts and login securely with Firebase Authentication integration

Donor Search

Find donors by blood type, location, and availability status

Health Monitoring

Track health status, eligibility, and get personalized health tips

Emergency Services

Access blood banks, emergency contacts, and quick-dial functionality

Donation Tracking

Track donation history with charts and real-time status updates

Smart Notifications

Receive alerts for blood requests and donation eligibility
